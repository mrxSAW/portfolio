# 🌐 Portfolio Personnel - ARQAS MOHAMED

## 📖 Description

Ce projet est un **Portfolio Personnel** développé en HTML, CSS et JavaScript.

Il a pour objectif de présenter mon profil professionnel, mes compétences techniques ainsi que mes différents projets réalisés dans le domaine du développement informatique.

Le portfolio sert également de vitrine numérique permettant aux recruteurs, enseignants et clients de découvrir mon parcours et mes réalisations.
